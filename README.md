# AcolheSer — Sistema de Gestão de Doações

Sistema web para gerenciamento de doações de materiais de construção 
e acompanhamento voluntário da ONG AcolheSer, desenvolvido como 
Projeto Integrador Extensionista (PIE I) do curso de ADS — Unimar.

## Descrição da ONG
A AcolheSer é uma ONG sediada em São Paulo, SP, que atua na coleta 
e distribuição de materiais de construção para comunidades em 
situação de vulnerabilidade, além de oferecer acompanhamento 
psicológico voluntário.

## Tecnologias
- Frontend: React + Vercel
- Backend: Node.js + Express + Render
- Banco de dados: MySQL + Railway
- Autenticação: JWT

## Estrutura do projeto
- /docs — Documento de Visão e documentação do projeto
- /backend — API REST (Node.js + Express)
- /frontend — Interface web (React)
- /database — Scripts SQL e migrations

## Instalação
1. Clone o repositório
2. Copie `.env.example` para `.env` e preencha as variáveis
3. Execute `npm install` em `/backend` e `/frontend`
4. Execute `npm start` em cada pasta
